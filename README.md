# zkclarkin.com

Personal website built with Astro and deployed to GitHub Pages at https://zkclarkin.com

## Quick Start

```bash
npm install          # Install dependencies
npm run dev          # Start dev server (http://localhost:4321)
npm run build        # Build for production
```

## Adding Content

Create a new `.astro` or `.md` file in `src/pages/` to add a page. The file will automatically become a route.

## Deployment

Push to GitHub and GitHub Actions will automatically deploy to `https://zkclarkin.com`.
